📖 什么是五度圈？

五度圈（Circle of Fifths）是音乐理论中最核心的视觉工具。它把 12 个调性 按 纯五度（7 个半音）的间隔排列成一个圆环，一眼就能看出调号、和弦关系、以及如何在不同调之间转换。

💡 核心概念： 顺时针走 = 升号方向（属调），逆时针走 = 降号方向（下属调）。 相邻的调最"亲近"（只差一个升/降号），切换最自然；对面的调最"远"。

👀 怎么读五度圈？

一、外圈 = 大调（Major Keys）

顶端是 C 大调（0 个升/降号），顺时针每走一格加一个升号，逆时针每走一格加一个降号。

方向顺序调号变化
🔄 顺时针C → G → D → A → E → B → F♯ → C♯升号 0 → 1 → 2 → … → 7
🔄 逆时针C → F → B♭ → E♭ → A♭ → D♭ → G♭ → C♭降号 0 → 1 → 2 → … → 7

🔑 巧记升号顺序： F-C-G-D-A-E-B（父吃瓜的爱饼）
🔑 巧记降号顺序： B-E-A-D-G-C-F（升号倒过来念）

二、内圈 = 关系小调（Relative Minor Keys）

每个大调都有一个 关系小调，它们共享完全相同的调号（升降号数量一样）。 从大调往顺时针方向数 3 格，就是它的关系小调。

大调CGDAEB
关系小调AmEmBmF♯mC♯mG♯m

* 简单记法：大调往下数 3 个半音 = 关系小调。比如 C 往下 3 个半音 → A → Am

三、底部 = 等音调（Enharmonic Keys）

圈底部的 F♯ = G♭、C♯ = D♭、B = C♭，它们是 等音调——键盘上弹的键完全一样， 只是写法不同。你可以选择升号少的那个来读谱更方便。

🛠️ 如何使用五度圈？

🎯 用法一：快速查调号

看到一首曲子，想知道它有几个升号/降号？看五度圈外圈的数字就行。

  • D 大调 → 看 D 的位置 → ♯2（两个升号：F♯、C♯）
  • E♭ 大调 → 看 E♭ 的位置 → ♭3（三个降号：B♭、E♭、A♭）

🎸 用法二：和弦进行

最常见的 I–IV–V 和弦进行就在五度圈上紧挨着。

  • 以 C 为例：I = C，IV = F（逆时针 1 格），V = G（顺时针 1 格）
  • C → F → G （无数歌曲的基础！）

🔄 用法三：转调（Modulation）

要在曲子中间换调？往相邻的调转最自然。

  • C → G（顺时针 1 格）：很流畅
  • C → D（顺时针 2 格）：有点张力
  • C → F♯（对面 6 格）：大胆的跳跃

🎼 用法四：找关系大小调

想给大调曲子加一段小调的色彩？直接用它的关系小调！

  • C 大调 → Am（共享 0 个升降号）
  • G 大调 → Em（共享 1 个升号）
  • 即兴 solo 时在这两个调间切换超好用

🎹 用法五：移调（Transposition）

曲子原调太高唱不上去？用五度圈快速找到新调。

  • C 调想降两个调 → 逆时针走 2 格 → B♭ 调
  • 所有和弦统一沿圈移动相同格数即可

🎵 用法六：借调和弦（Borrowed Chords）

想丰富和弦色彩？从相邻调"借"和弦。

  • C 大调里突然用 B♭ 和弦（来自 F 调）→ 流行/摇滚常用
  • 关系小调的和弦也可以借 → 同理

🎹 实战示例

示例：用五度圈分析 C 大调的常用和弦

• I 级（主和弦） = C（圈中 C 的位置）

• V 级（属和弦） = G（顺时针 1 格 → 紧张，渴望回到主和弦）

• IV 级（下属和弦） = F（逆时针 1 格 → 温暖，稳定的过渡）

• vi 级（关系小调） = Am（C 的关系小调 → 抒情的色彩）

→ 经典 I–V–vi–IV 进行：C → G → Am → F（无数流行歌的基础！）

🎸 吉他手/钢琴手实用技巧

把五度圈画在琴房墙上。即兴 jam 时，用五度圈上相邻 3~4 个调的和弦随意组合，几乎不会"跑调"。 试试这组：C → Am → F → G（四个和弦都在五度圈 C 的邻近位置）。
