📖 什么是五度圈？

五度圈把 12 个调性 按纯五度（7 个半音）间隔排成圆环。 相邻的调最亲近（只差 1 个升/降号），对面的调最远。

💡 核心：顺时针 = 升号方向 | 逆时针 = 降号方向 | 外圈 = 大调 | 内圈 = 关系小调（共享相同调号）
五度圈最大的用处：找到任何调的 I–ii–iii–IV–V–vi–vii° 七级和弦 → 点击圈上的键试试！

🛠️ 核心技能：一眼找出任何调的各级和弦

这是五度圈最实用的功能。给定一个调，它的 7 个顺阶和弦在圈上排成一段连续的弧：

以 C 大调为例：

IV = F ← 逆1 ← I = C → 顺1 → V = G → 顺1 → ii = Dm → 顺1 → vi = Am → 顺1 → iii = Em → 顺1 → vii° = Bdim

🟢 大三和弦 | 🔴 小三和弦 | 🟣 减三和弦 | 🔵 主和弦(I)

记忆规律

级数圈上位置（从 I 出发）和弦性质举例（C大调）
I（主）本位大三C
IV（下属）逆时针 1 格大三F
V（属）顺时针 1 格大三G
ii顺时针 2 格小三Dm
vi顺时针 3 格小三Am
iii顺时针 4 格小三Em
vii°顺时针 5 格减三Bdim

💡 规律：I、IV、V 是大三和弦（紧挨在一起的 3 个）；ii、iii、vi 是小三和弦（顺时针方向 2/3/4 格）；vii° 是减三和弦。 这个规律对任何大调都适用！选 C 调记熟，换到其他调就依葫芦画瓢。

👀 怎么读五度圈？

外圈：12 个大调

顶端 C（0 升/降号），顺时针加升号，逆时针加降号。

顺时针（升号）CGDAEBF♯C♯
逆时针（降号）CFB♭E♭A♭D♭G♭C♭

🔑 升号顺序口诀：F-C-G-D-A-E-B（父吃瓜的爱饼）
🔑 降号顺序口诀：B-E-A-D-G-C-F（升号倒过来）

内圈：12 个关系小调

每个大调顺时针走 3 格 = 关系小调。如 C → 顺3 → A → Am。大调往下数 3 个半音也一样。

底部：等音调

F♯=G♭、C♯=D♭，键盘上一样但写法不同，选升/降号少的读谱更轻松。

🎯 更多用法

🎯 查调号

看一眼圈上的数字就知道几个升降号。D大调 = ♯2（F♯, C♯），E♭大调 = ♭3（B♭, E♭, A♭）。

🔄 转调

相邻调切换最自然（差1个升降号）：C→G 很顺畅。
对面调切换最大胆（差6个）：C→F♯ 很刺激。

🎹 移调

唱不上去？统一把整首歌的所有和弦沿圈移动相同格数。C 降两格 → B♭ 调。

🎵 借调和弦

从相邻调借和弦增加色彩。C大调里突然用 B♭（来自F调）→ 摇滚/流行标配。
